
# Rostering Dashboard (Beta)

A react app POC that shows rostering of sample employees

## Installation
1. clone this repo then

```bash
  cd [repo_name] && npm i
```

2. run the app
```bash
  npm run start
```

3. Enjoy
## Tech Stack/Features
- Node >= 16
- react-cra
- react-big-calendar
- swiper
